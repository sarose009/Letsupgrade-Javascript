var is global variable which is declared in a whole block of code. It is functional scope.
let is local variable which is declared inside a particular scope. It is block scope.
const as its name is used when we want to declare constant value and it cannot be changed but can be modified in object.
Examples:

var


var name = "Saroj" var age = "24" var language = ['english','nepali','hindi']; var friends = { name:"diwash", age:"21", } var a; var a= null;

let


{ let name = "sai sir"; }

constant


const language= "english"; const fruits = ['mango','banana']
